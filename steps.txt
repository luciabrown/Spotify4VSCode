If the pause/play functionality does not work, delete token.json and run the application again


Docker build: docker build -t luciafbrown/project-full:latest -f Dockerfile ..

Use baked-in .env (CI secrets):
docker run --name project-full -p 12345:12345 luciafbrown/project-full:latest

or
Use local .env file at runtime
docker run --name project-full -p 12345:12345 --env-file ./backend/.env luciafbrown/project-full:latest

docker start project-full
